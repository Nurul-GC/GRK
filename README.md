# GRK - Gerenciamento de Reservas Kasper

    Projeto de um sistema de gerenciamento de restaurante, desenvolvido em PHP/Laravel, utilizando o paradigma de orientação a objetos.
    O sistema é projetado para facilitar a gestão de pedidos, reservas de mesas e cardápios, proporcionando uma experiência eficiente tanto para 
    os clientes quanto para a administracao do restaurante.

## Funcionalidades

- Reservas de Mesas: Os clientes podem reservar mesas para datas e horários específicos, garantindo uma experiência personalizada.
- Gerenciamento de Cardápio: O sistema permite aos administradores do restaurante gerenciar o cardápio, adicionando, editando ou removendo itens conforme necessário e ainda mantem a gestao dos pedidos e reservas feitas pelos clientes.
